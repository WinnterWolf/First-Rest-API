Type /app/rest at the end of the URL to find all entries (HTTP GET)

On Postman Type /app/rest/add at the end of the URL to add an entry in JSON(HTTP POST)

Type /app/id at the end of the URL and the id number you want to find (HTTP GET)

On Postman Type /app/rest/change/id at the end of the URL and the id number you want to change + body in JSON format (HTTP PUT)

On Postman Type /app/rest/delete/all at the end of the URL to delete all entries (HTTP DELETE)
